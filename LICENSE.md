### Applying Config Changes via Bash (Pre-execution)

If you are managing multiple config profiles, you can use the following bash snippet to swap configs before launching:

```bash
#!/bin/bash
# Sailor Piece Script — Config Profile Switcher

PROFILE=$1
CONFIG_DIR="./configs"
TARGET="./sailor_piece_active.lua"

if [ -z "$PROFILE" ]; then
  echo "Usage: ./switch_config.sh [aura|boss|dungeon|pvp]"
  exit 1
fi

if [ -f "$CONFIG_DIR/$PROFILE.lua" ]; then
  cp "$CONFIG_DIR/$PROFILE.lua" "$TARGET"
  echo "✅ Active config switched to: $PROFILE"
else
  echo "❌ Profile '$PROFILE' not found in $CONFIG_DIR"

fi
```

## 🔧 Troubleshooting & Common Issues

Even a rock-solid script occasionally hits friction. Here are the most common issues users report and their verified solutions.

| Issue | Likely Cause | Fix |
|---|---|---|
| GUI does not appear after execute | Injected too early (game still loading) | Wait for full spawn screen, re-inject |
| Auto Farm stops after a few minutes | Server lag or mob spawn changes | Toggle Auto Farm off and back on |
| Kill Aura not registering hits | Radius too low for current area | Increase radius to 40–50 in CONFIG |
| Teleport puts character inside wall | SafeWarp disabled | Enable `SafeWarp = true` in config |
| Script crashes on mobile after 10 mins | Insufficient device RAM | Close background apps, free up memory |
| God Mode stops working after update | Game patch changed damage validation | Pull latest script version from repo |
| Server Hop not triggering | HopDelay set too high | Lower `HopDelay` to 30–40 seconds |
| Aura Chest count not increasing | Wrong server region | Enable Server Hop; try different server |
| Script loads but no modules visible | GUI off-screen on small displays | Drag the GUI panel toward screen center |
| Executor detaches randomly | Roblox update changed injection point | Update executor to latest version |

> 🆘 If none of the above resolves your issue, open a GitHub Issue with your platform, executor version, and a description of the problem. The community typically responds within 24 hours.
